## Summary

API Surface is the contract layer for HTTP services: OpenAPI, operation semantics, workflow descriptions, extensions, versioning, and event delivery. Use it when agents need to discover or generate calls against a web API without guessing from prose.

- **Owns:** HTTP APIs, OpenAPI 3.1, Arazzo workflows, webhooks, API versioning, and machine-readable service contracts.
- **Does not own:** cross-framework tool registry design; see [Tool Design](/docs/tool-design). MCP protocol servers; see [MCP Servers](/docs/mcp-servers).
- **Agent-ready means:** complete schemas, stable operation IDs, explicit safety/idempotency semantics, structured errors, and versioned behavior.
