## Summary

Context Files are repository and editor instructions for coding agents working inside a codebase. Use them to document commands, boundaries, architecture cues, package scopes, and tool-specific overlays.

- **Owns:** `AGENTS.md`, `CLAUDE.md`, Cursor rules, Copilot instructions, nested monorepo context, and drift detection.
- **Does not own:** public web discovery files like `llms.txt`, `robots.txt`, or JSON-LD; see [Discovery](/docs/discovery).
- **Agent-ready means:** agents can start safely, run the right commands, follow local conventions, and know when to ask before risky work.
