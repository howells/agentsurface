## Summary

Discovery covers public, web-facing signals that help agents find, read, cite, and govern access to your docs, APIs, and capability metadata. Use it for sites, hosted documentation, public API catalogs, and crawler-facing policy.

- **Owns:** `llms.txt`, structured data, Markdown content negotiation, `robots.txt`, Content Signals, `.well-known` metadata, and public content structure.
- **Does not own:** repository/editor operating instructions; see [Context Files](/docs/context-files). MCP implementation details; see [MCP Servers](/docs/mcp-servers).
- **Agent-ready means:** agents can find canonical docs and machine-readable contracts, retrieve token-efficient content, and respect explicit access policy.
